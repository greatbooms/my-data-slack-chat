# 에이전트 안내

이 폴더는 관리자 워크스페이스 GraphQL 기능의 통합 테스트를 둡니다.

## 확인 포인트

- 테스트는 관리자 세션 로그인 후 `/admin/graphql`을 호출합니다.
- 워크스페이스 생성, 수정, 소프트 삭제, 복구 응답을 검증합니다.
- 삭제된 워크스페이스는 기본 목록에서 제외되고, `includeDeleted: true`일 때만 확인 가능해야 합니다.
- 삭제된 워크스페이스에는 새 데이터소스를 만들 수 없어야 합니다.
- 워크스페이스 삭제 정책을 바꾸면 데이터소스 목록, 수동 수집, 검색 범위 테스트도 함께 확인합니다.
