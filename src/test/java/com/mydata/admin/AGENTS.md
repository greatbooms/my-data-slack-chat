# 에이전트 안내

이 폴더는 관리자 세션 인증, 관리자 GraphQL, 관리자 UI 정적 서빙 테스트를 둡니다.

## 확인 포인트

- 관리자 기능은 `/admin/auth/login`으로 세션을 만든 뒤 `/admin/graphql`로 호출합니다.
- 로그인과 GraphQL mutation 요청에는 CSRF 토큰이 필요합니다.
- 관리자 권한이 없는 세션은 관리자 GraphQL 기능을 사용할 수 없어야 합니다.
- 데이터 변경 테스트는 소프트 삭제와 ACL 정책 갱신까지 함께 확인합니다.
- `/admin-ui/login`은 비로그인 사용자도 React 앱 shell을 받을 수 있어야 합니다.
- 보호된 `/admin-ui/**` 경로는 관리자 세션이 없으면 로그인으로 이동해야 합니다.
